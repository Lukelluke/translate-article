#### 模块

1. ``__name__``属性

   每个模块都有一个``__name__``属性，当其值是``__main__``时，表明该模块自身在运行，否则是被引入。

2. 子包，文件夹下必须存在``__init__.py``文件，内容可以为空，用于标识当前文件夹是一个包。

   ```python
   if __name__ == '__main__':
   	print （“作为主程序运行”）
   else:
   	print （”初始化“）
   ```

3. ``__name__``属性

   每个模块都有一个``__name__``属性，当其值是``__main__``时，表明该模块自身在运行，否则是被引入。
